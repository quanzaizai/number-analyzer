# Number Analyzer

## 项目说明
这是一个使用 Python 编写的数字统计小项目。

用户输入一串整数（用空格隔开），程序会对这组数字进行统计分析，并支持循环输入、异常处理以及输入 `q` 退出程序。

这是当前“做中学”阶段完成的第一个可运行、可迭代、可展示的小项目。

## 当前功能
- 输出原始数字列表
- 统计数字总个数
- 统计所有数字总和
- 统计偶数个数
- 统计奇数个数
- 统计大于 10 的数字个数
- 统计大于 10 的数字总和
- 统计平均值
- 处理空输入
- 处理非法输入
- 支持输入 `q` 退出程序
- 支持循环运行
- 支持输入`h`查看历史功能
- 支持将统计结果保存到本地文件`history.txt`

## 运行方式

```bash
uv run main.py
```

## 运行示例
### 1. 正常输入：
```text
请输入一串数字：3 4 5 6 7 8
总和为：33
总数的个数为：6
偶数个数为：3
奇数个数为：3
大于10的数的总和为：0
大于10的数的个数为：0
这些数的原始列表为：[3, 4, 5, 6, 7, 8]
这些数的平均数为：5.5
```


### 2. 空输入：
```text
请输入一串数字：
请至少输入一个整数
```

### 3. 非法输入：
```text
请输入一串数字：3 hello 4 6
请输入正确的整数数据
```

### 4. 输入 q 退出：
```text
请输入一串数字：q
程序结束
```

### 5. 输入 h 的情况:
```
请输入一串数字：h
当前还没有历史记录

请输入一串数字：h
（显示历史记录内容）
```
